Hexo与GitHub搭建个人博客

第一步 ： 下载nodejs ，npm和git

1.在官网下载nodejs ，下载后添加到环境变量，方法如下

首先打开nodejs所在的文件夹，创建一个名为node_cache的文件夹和一个名为 node_global的文件夹。在node_global文件里新建一个名为node_modules的文件夹

在导航栏搜索编辑环境变量，打开后按 高级-环境变量-系统变量-新建，变量名设为NODE_PATH，变量值设为node_modules的路径

接着点击用户变量中的Path，选择编辑，将默认的 C 盘下【 AppData\Roaming\npm 】修改成 【node_global】的路径，点击确定，在系统变量中选择Path，编辑然后添加NODE_PATH，确定，确定。

配置完成后在cmd中运行以下代码来检验成果

npm install -g express
